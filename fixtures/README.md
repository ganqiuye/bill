# 测试数据说明

## simple-bills（推荐入门）

| 文件 | 说明 |
|------|------|
| `simple-bills.csv` | 2 行，最快验证导入与调账 |
| `simple-bills.xlsx` | 同上，Excel 格式 |

| 产品 | 实际金额 | 目标金额 | 差额 | 状态 |
|------|----------|----------|------|------|
| 产品A | ¥5,000 | ¥4,500 | +¥500 | 盈余 |
| 产品B | ¥3,000 | ¥3,500 | -¥500 | 亏空 |

**建议调账：** 从产品A 调出 ¥500 → 调入 产品B，原因填「结构调平」。调平后两产品差额均为 0。

导入列映射：`产品` → 产品，`金额` → 金额，`客户` → 客户，`目标` → 目标金额，`账期` → 账期。

## sample-bills（多产品）

| 文件 | 说明 |
|------|------|
| `sample-bills.csv` / `sample-bills.xlsx` | 4 行，A/B 公司各 1 盈余 + 1 亏空 |

可用于测试「产品」「客户」两种维度的看板与对比。
